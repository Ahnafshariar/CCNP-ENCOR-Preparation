# IOU Web Console — Lab Environment

Notes on using the **Cisco IOU (IOS on Unix) Web** console for quick CLI practice and verification alongside EVE-NG.

## When I use it
- Fast, lightweight L2/L3 feature drills without spinning up a full EVE-NG topology
- Repeating verification commands while writing lab docs

## Workflow
1. Launch the IOU Web console
2. Build / load the device set
3. Apply config, run the verification commands documented in each lab
4. Capture output → paste into the lab's `README.md` under **Verification**

> Same rule as EVE-NG: no Cisco IOU binaries are stored in this repo.
